¿Cómo funciona Survey?

El sistema de consulta Surbey garantiza los tres pilares fundamentales de un sistema de votación: anonimato, trazabilidad y control del voto.

Control del voto

Se crea una marca de tiempo vinculará a cada usuario en el momento de realizar la votación, garantizando que este no se podrá emitir dos veces.

Anonimato

Para garantizar el anonimato se emplean dos tablas separadas, una contiene los usuarios, la segunda los votos realizados vinculados al uuid. Una vez se realiza el voto, el uuid de la tabla de usuarios es eliminado, haciendo imposible identificar qué usuario ha realizado cada uno de los votos.

Trazabilidad

A cada usuario se le proporciona un uuid al iniciar la votación, con él, es posible consultar tu votación para asegurar de que este se ha registrado de forma correcta. De otra parte los datos (anonimizados) son públicos, por lo que es posible auditar el proceso completo durante todo el proceso.
